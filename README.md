# Landing Page: เรียนดวงกับบรมครูโหร พัฒนา พัฒนศิริ

ไฟล์หลักอยู่ที่ `index.html` (หน้าเดียว) พร้อม `styles.css` และ `script.js`

## วิธีเปิดใช้งาน (ง่ายที่สุด)

- ดับเบิลคลิก `index.html` เพื่อเปิดในเบราว์เซอร์

## วิธีเปิดแบบ Local Server (แนะนำ ถ้าฝังวิดีโอแล้วเจอข้อจำกัด)

ถ้าคุณมี Python:

```bash
python -m http.server 5173
```

แล้วเปิด `http://localhost:5173`

## ปรับแต่งเร็ว

- เปลี่ยนข้อความ/หัวข้อ: `index.html`
- เปลี่ยนสี/เลย์เอาต์: `styles.css`
- เมนูมือถือ + แท็บเส้นทางเรียน: `script.js`
